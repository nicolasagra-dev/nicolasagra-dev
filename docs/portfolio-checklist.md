# Checklist de portfolio para estagio

Use este checklist quando publicar ou revisar projetos no GitHub.

## README de cada projeto

- Problema resolvido em 2 ou 3 linhas.
- Print, grafico ou GIF logo no inicio.
- Stack usada.
- Como executar localmente.
- Resultados principais.
- Limitacoes e proximos passos.
- Link do app ao vivo quando existir.

## Metadados do GitHub

- Description preenchida.
- Website preenchido quando houver deploy.
- Topics com tecnologias e area do projeto.
- Repositorios principais fixados no perfil.
